# 创新点

本程序为lz小学生时精心纯手工打磨代码，展示了lz卓越的拖动控件与点击编译的技术，以及在那个没法随意用电脑的年纪苦中作乐的态度（本项目原本由LZ在课间创作，在**草稿纸**上编译与运行）。

现在看来，本程序在这个ai时代能给人一种纯真的感觉。如今让他在GitHub上重现人世，震撼首发！

# 功能

用VB控件模拟出一个系统：Windows SB，附带一些小工具

# 用法

双击运行即可，你打开了就知道要怎么点了
